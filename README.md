# 🎬 CineMatch AI

## Discover Your Next Favorite Movie

CineMatch AI is an intelligent hybrid movie recommendation system that combines:

- Collaborative Filtering
- Content-Based Filtering
- Hybrid Recommendation Engine
- Mood-Aware Ranking
- Time-Aware Recommendations
- Natural Language Movie Queries

Built using Python, Streamlit, Scikit-Learn and TMDb API.

---

## Features

### 🎯 Personalized Recommendations
Generates movie suggestions based on user rating history.

### 🧠 Hybrid AI Engine
Combines collaborative filtering and content-based filtering.

### 🎭 Cinematic Mood Selection
Choose moods like:

- Mind-Bending
- Dark
- Romantic
- Funny
- Psychological
- Epic

### 🤖 AI Movie Concierge

Examples:

- Recommend a dark crime thriller
- I want a mind-bending sci-fi movie
- Funny movies for tonight

### ⏰ Time-Aware Recommendations

Recommendations adapt according to:

- Morning
- Afternoon
- Evening
- Night

### 🎬 Trailer Integration

Watch trailers directly inside the application.

### 📊 Recommendation Quality Score

Evaluates recommendation quality using:

- Average Rating
- Popularity
- User Similarity
